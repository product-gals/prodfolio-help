Prodfolio includes a rich text editor for longer-form sections like case study SIGNAL steps and your product philosophy.

## Formatting options

The editor toolbar includes:

- **Bold** (Ctrl/Cmd + B)
- **Italic** (Ctrl/Cmd + I)
- **Underline** (Ctrl/Cmd + U)
- **Heading** (H3 level)
- **Bullet list**
- **Numbered list**
- **Links** — Select text and click the link icon to add a URL

## Character limits

Each text field has a character limit based on the section. The editor displays a character count as you type, with a visual warning as you approach the limit.

Character counts are based on text content only — formatting (bold, italic, etc.) doesn't count against the limit.

## Tips

- Use headings and lists to break up longer sections and improve readability
- Links automatically open in a new tab for visitors
- If you paste formatted text from another source, the editor preserves supported formatting and strips unsupported styles
