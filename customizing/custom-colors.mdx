<Note>Custom colors are a Pro feature. [Upgrade to Pro →](/billing/plan-comparison)</Note>

Pro users can fully customize their portfolio's color palette using hex codes.

## Customizable colors

- **Primary** — Main accent color (buttons, links)
- **Secondary** — Supporting color
- **Accent** — Highlight color for emphasis
- **Background** — Page background color
- **Text** — Body text color

## How to set custom colors

1. Open the portfolio builder
2. Scroll to the **Theme** section
3. Select any preset as a starting point
4. Click on a color field and enter a hex code (e.g., `#7C3AED`)
5. Colors save automatically

## Contrast ratios

Prodfolio checks the contrast ratio between your text and background colors to help ensure readability. If contrast is too low, you'll see a warning.

## Resetting to a preset

Click **Reset to Preset** to revert all custom colors back to the selected theme's defaults.
