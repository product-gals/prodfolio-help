# Free vs. Pro Analytics

| Feature | Free | Pro |
|---------|------|-----|
| Total view count | Yes | Yes |
| Week-over-week trend | Yes | Yes |
| Traffic sources breakdown | No | Yes |
| Case study rankings | No | Yes |
| Visitor actions (clicks, downloads) | No | Yes |
| Portfolio comparison | No | Yes |
