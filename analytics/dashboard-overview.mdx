# Analytics Dashboard Overview

Track who's viewing your portfolio and which case studies resonate most.

{/* TODO: Add video embed */}

## Accessing analytics

Go to **Analytics** in the sidebar navigation. If you have multiple portfolios, use the portfolio selector to filter by a specific one or view all.

## Timeframe

Use the timeframe selector to view data for:
- Last 7 days
- Last 30 days
- Last 90 days
- All time

## What you'll see

### All users
- **Total views** — How many times your portfolio has been viewed
- **Trend indicator** — Week-over-week change

### Pro users
- **Traffic sources** — Where your visitors are coming from
- **Case study rankings** — Which case studies get the most views
- **Visitor actions** — Clicks on email, downloads, and links
- **Portfolio comparison** — Compare performance across multiple portfolios

## No data yet?

If you see an empty state, it means either:
- Your portfolio isn't published yet — [publish it first](/publishing/publish-portfolio)
- Your portfolio hasn't received any views yet — [share your link](/publishing/sharing-your-link)
