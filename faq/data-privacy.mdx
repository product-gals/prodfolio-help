## What's public

Only your **published portfolio** is visible to others. This includes the sections you've filled in: basics, about, experience, case studies, references, beyond work, skills, and tools.

## What's private

- **Draft portfolios** — Not visible to anyone but you
- **Internal portfolio descriptions** — Only visible in your dashboard
- **Analytics data** — Only visible to you
- **Account settings** — Only visible to you
- **Access management history** — Only visible to you
- **Password-protected case studies** — Only visible to people you grant access to

## Data security

- All data is stored securely in Supabase (PostgreSQL)
- Authentication uses industry-standard protocols (including OAuth PKCE for Google login)
- HTML content is sanitized to prevent XSS attacks
- We never log personally identifiable information (PII)

For full details, see our [Privacy Policy](https://prodfolio.io/privacy).
