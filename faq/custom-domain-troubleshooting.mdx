# Custom Domain Troubleshooting

## Domain stuck on "Pending"

DNS changes can take up to 48 hours to propagate. If your domain is still pending after 48 hours:

1. Verify your CNAME record is set correctly in your registrar
2. Make sure there are no conflicting A records for the same subdomain
3. Try clicking **Verify** again in the portfolio builder

## Domain showing "Error"

- Double-check that your CNAME value matches exactly what Prodfolio provided
- Ensure there are no typos in the record name or value
- Some registrars add the root domain automatically — if your domain is `portfolio.janedoe.com`, you may only need to enter `portfolio` as the record name

## Using a root domain

If you want to use a root domain (e.g., `janedoe.com` without a subdomain), some registrars don't support CNAME records at the root. Check if your registrar supports **ALIAS** or **ANAME** records instead.

## Still having issues?

Contact us at **support@prodfolio.io** with your domain name and a screenshot of your DNS settings, and we'll help you get set up.
